# Noctune — Product Design & UX Philosophy

## Product Vision

Noctune is a **procedural ambient sound platform** — every sound is generated in real-time by the Web Audio API, not played from audio files. This means zero download size for audio, infinite variation, and the ability to layer unlimited combinations.

**Core premise:** Replace the noise of the world with intentional, curated soundscapes. A sanctuary for focus, sleep, and calm.

**Target user:** Knowledge workers, students, meditators, and anyone who uses ambient sound to concentrate or relax. Desktop-first (Spotify-grade layout) with full mobile support.

### Design Principles

1. **Invisible audio** — the UI should fade away; the sound is the product. Ambient overlay mode hides the entire UI.
2. **Spatial depth** — the 3D hero isn't decoration; it's a navigable sound preview. Each orb = a soundscape.
3. **Desktop-first, mobile-competent** — the full feature set exists on desktop. Mobile gets the same functionality through a slide-in sidebar, bottom sheet, and bottom nav.
4. **Zero friction** — no accounts required to play sounds. No loading spinners for audio. Instant playback on click.
5. **Procedural over recorded** — every sound is generated by oscillators, noise buffers, and LFOs. This gives infinite, non-repeating soundscapes.

### Glassmorphism

The `.glass` class provides the app's signature frosted-glass look:

```css

  background: rgba(255,255,255,0.03);
  backdrop-filter: blur(16px);
  border: 1px solid rgba(255,255,255,0.08);

```

Used for: cards, dropdowns, modals, sidebar, topbar (when scrolled), sound picker, timer dropdown.

## UX Patterns

### Playback Flow

1. User clicks **SoundCard** or **Environment pill**
2. Sound starts immediately (no loading state — audio is procedural)
3. **PlayerBar** appears at the bottom with playback controls
4. **Equalizer** icon animates on active sounds
5. User can layer more sounds on top (up to 16 concurrent)
6. **RightPanel** shows all active layers with individual volume control

### Navigation Flow

- **Desktop:** Sidebar (persistent) + TopBar + RightPanel
- **Mobile:** BottomNav (always visible) + Hamburger (opens sidebar) + Search (⌘K)

### Sleep Timer Flow

1. User clicks Timer icon in PlayerBar
2. Dropup shows preset options: 15m, 30m, 45m, 60m, 2h
3. Timer starts countdown; button shows remaining time in accent color
4. At 0: sounds **fade out over 30 seconds** (not abrupt stop)
5. Timer auto-cancels if all sounds stop manually

### Search Flow

1. Trigger: ⌘K shortcut or click search bar in TopBar
2. Full-screen overlay opens with auto-focused input
3. Type shows autocomplete suggestions from 31 keyword categories
4. Press Enter or click suggestion to search
5. Results shown as SoundCard grid; clicking a result plays it and closes search
6. Recent searches saved (max 10) via zustand persist

### Favorites Flow

1. Click Heart icon on any SoundCard
2. Toast confirms "Added to favorites" (not shown yet — toast is future)
3. Heart fills red (accent-red)
4. Sound appears in `/favorites` page
5. Click Heart again to remove

### Ambient Mode Flow

1. Play one or more sounds
2. Click Moon icon in PlayerBar
3. Full-screen dark overlay fades in
4. UI auto-hides; cursor disappears after 3s
5. Space toggles play/pause; volume slider visible
6. ESC or click-outside to exit

## Accessibility

- All icon buttons have `aria-label`
- Keyboard navigation: Tab through all interactive elements
- Focus-visible rings on all interactive elements: `outline: 2px solid accent/40`
- `prefers-reduced-motion` support: disables all animations and transitions
- WCAG AA contrast: text-primary is `#F8FAFC` on `#090B12` (14.5:1 ratio)
- Reduced motion setting in Settings → Appearance

## Data Persistence

Data is persisted to `localStorage` via zustand `persist` middleware:

| Store | Key | Persisted Data |
|-------|-----|----------------|
| `useUiStore` | `noctune-ui-storage` | sidebarOpen, rightPanelOpen, recentSearches |
| `useSettingsStore` | `noctune-settings-storage` | theme, crossfade, timer, reducedMotion, etc. |
| `useFavoritesStore` | `noctune-favorites-storage` | soundIds, collectionIds, savedCollections |
| `useMixerStore` | `noctune-mixer-storage` | layers, presets, masterVolume |
| `useAudioStore` | Not persisted | AudioContext can't be serialized |
| `useToastStore` | Not persisted | Ephemeral UI state |
